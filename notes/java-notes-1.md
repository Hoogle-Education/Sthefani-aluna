# Sequencial vs. Binary

## Array acess

` <nameOfArray> [column] `

## Array 2D / matrix acess

` <nameOfArray>[<row>][<column>] `

**test1: printing `array[i][j]`**
output:
```
2 3 1 
1 2 3
4 3 2
print matrix:
[0][0] =2 [0][1] =3 [0][2] =1 
[1][0] =1 [1][1] =2 [1][2] =3 
[2][0] =4 [2][1] =3 [2][2] =2 
```

**test2: printing `array[j][i]`**
output2:
```
1 2 3
3 2 1
3 4 5
print matrix:
[0][0] = 1 [1][0] = 3 [2][0] = 3 
[0][1] = 2 [1][1] = 2 [2][1] = 4 
[0][2] = 3 [1][2] = 1 [2][2] = 5 
```
